# Nutrition RAG Based Application

A Retrieval-Augmented Generation (RAG) based personalized nutrition advisor that provides evidence-based diet recommendations using trusted sources. The system generates tailored meal suggestions, nutrient insights, and health warnings based on user conditions such as deficiencies, dietary preferences, and medical needs.

## Getting Started

### Prerequisites

- Python 3.11.8
- Pip

### Installation

1. **Create a virtual environment:**
   ```bash
   python -m venv venv
   ```

2. **Activate the virtual environment:**
   ```bash
   source venv/bin/activate
   ```

3. **Install the required dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

## Usage

To use the application, run the `main.ipynb` notebook.
